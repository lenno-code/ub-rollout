Requirements
============

This section contains all project requirements.

System Requirements
-------------------
   

.. req:: Windshield Wiper Speed Control
   :id: REQ_001
   :status: open
   :priority: high
   
   The system shall control the front windshield wiper speed based on rain intensity

.. req:: Windshield Wiper manual override
   :id: REQ_002
   :status: open
   :priority: high

   The system shall allow manual override of the wiper speed by the driver

.. req:: Windshield safety fallback
   :id: REQ_003
   :status: open
   :priority: high

   The system shall ensure safe operation in case of sensor failure
