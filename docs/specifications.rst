Specifications
==============

Technical specifications derived from requirements.
   

.. spec:: Three Speed modes
   :id: SPEC_001_001
   :status: open
   :implements: REQ_001

   The system shall support at least three wiper speed levels (low medium high)

.. spec:: Automatic Speed adaptation
   :id: SPEC_001_002
   :status: open
   :implements: REQ_001

   The system shall automatically adjust speed based on sensor input

.. spec:: Select wiper mode
   :id: SPEC_002_001
   :status: open
   :implements: REQ_002

   The driver shall be able to select wiper speed via a control lever

.. spec:: Wiper mode fallback
   :id: SPEC_003_001
   :status: open
   :implements: REQ_003

   The system shall default to intermittent mode if the rain sensor fails
